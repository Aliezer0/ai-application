📖 项目简介

本项目是一个面向企业内部的智能知识库问答系统，结合了检索增强生成（RAG）与 Agent 技术。系统能够自动处理多源文档（如 PDF），通过向量检索与大语言模型结合，实现精准的自动化问答；同时扩展为具备工具调用能力的智能助手，可自主决定调用知识库、网络搜索、计算器等工具，处理复杂查询任务，有效提升信息获取效率。
✨ 功能特点
    RAG 精准问答：基于文档内容检索，生成严格基于上下文的答案，有效减少模型幻觉。
    多源文档支持：支持 PDF、网页等常见文档格式，可扩展更多类型。
    智能 Agent 集成：将 RAG 链包装为工具，与搜索、计算器等工具组合，实现多步推理任务（如结合内部知识库与实时信息）。
    混合检索优化：结合向量检索与关键词检索，提升召回率；支持分块大小、重叠等参数调优。
    可扩展性强：模块化设计，易于添加新工具或替换模型。

🛠️ 技术栈
    核心框架：LangChain, LangChain-Classic
    大语言模型：DeepSeek API（兼容 OpenAI 格式）
    向量数据库：Chroma + HuggingFace Embeddings
    工具：DuckDuckGo Search, 自定义计算器
    开发语言：Python 3.10
